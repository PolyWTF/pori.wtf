ふとブログを作ってみたいと思ったので以前から気になっていたNetlify + Gatsbyでブログを構築してみることにしました。

##環境構築

install Gatsby  
`yarn global add Gatsby`

Gatsbyのテンプレートを作る  
`gatsby new folderName https://github.com/gatsbyjs/gatsby-starter-blog`
